サンプルの実行方法

1. sample.TranByFileMainクラスのmainメソッドを実行すると、Bean定義ファイルを使用した宣言的トランザクションのサンプルが実行されます。

2. sample.TranByAnnotationMainクラスのmainメソッドを実行すると、アノテーションを使用した宣言的トランザクションのサンプル(有効化をBean定義ファイルで行う)が実行されます。

3. sample.TranByJavaConfigMainクラスのmainメソッドを実行すると、アノテーションを使用した宣言的トランザクションのサンプル(有効化をJavaConfigで行う)が実行されます。

4. sample.TranByProgrammaticMainクラスのmainメソッドを実行すると、明示的トランザクションのサンプルが実行されます。
